About Me

The biggest way for me to stay motivated is think of a program that intrigues you but is just out of reach of your abilities. Do a step ladder of difficult programs that engage you. My best example, the one that really got me going, was my messages app. I knew python basics, but not enough to write anything significant. So I thought of something: an app that would query a website to check the status of something, and then send me a SMS based upon the result. From that, I had to learn how to interact with websites using JSON, and SMTP libraries for python. After I reached that goal, I actually just kept building on it. Now I wanted a GUI, so I learned a framework. I wanted to be able to interact with it via text message, so I learned IMAP libraries and parsing. I wanted a history of queries, so I learned to use a really simple database. I wanted to learn how to run it in the background, so I learned about services etc. Find a project that interests you that you can't do... and do it. line by line. Make it messy if you don't know where to start, and patch it up and streamline it later.
